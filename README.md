# IIT JEE Question Bank

Cloudflare migration target for the IIT JEE Question Bank app.

Source app: `arnabsenapati/tsv-to-gsheet` branch `dev`.

## Planned Cloudflare services

- Cloudflare Pages for the React/Vite frontend in `web/`
- Cloudflare Workers for `/api/*`
- Cloudflare D1 for question-bank data
- Cloudflare R2 for uploads, images, exports, and snapshots
